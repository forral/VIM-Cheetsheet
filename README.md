# VIM-Cheetsheet

## Change  Modes:
* `i` Insert Mode
* ´esc´ Command Mode
* Last Line Mode

## Last Line Mode
´:q!´ exit without saving
´:q´ exit when there's nothing changed to save
´:w´ save without quit
´:wq´ save and quit
´:wq [filename]´ save file, give it a name and quit 

## Command Mode
´u´ undo
´dd´ delete line
